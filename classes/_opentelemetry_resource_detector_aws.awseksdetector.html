Class AwsEksDetector

The AwsEksDetector can be used to detect if a process is running in AWS Elastic Kubernetes and return a Resource populated with data about the Kubernetes plugins of AWS X-Ray. Returns an empty Resource if detection fails.

See https://docs.amazonaws.cn/en_us/xray/latest/devguide/xray-guide.pdf for more details about detecting information for Elastic Kubernetes plugins
